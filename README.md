`NEYRONKA.py` - файл с кодом от нейросети, которая по жалобам определяет специальность врача, к которому
              обращались с похожими симптомами <br>
`MAP.py` - файл с кодом, в котором по адресу определяется местоположение человека и все мед. учреждения,
              находящиеся в радиусе 5 км, затем выбирается ближайшая больница и строится мрашрут от человека к ней <br>
`audio_in_text.py` - файл с кодом бота, который конвартирует ГС пользователя в текст <br>
`bot.py` - файл с кодом ля взаимодействия с пользователем через Telegram бота <br>
